dropthe-rust

Rust crate for DropThe open data platform. Type-safe async access to 1.83M entities and the knowledge graph.

About DropThe

This library provides idiomatic Rust bindings for the DropThe open data API. DropThe is a data utility media network that aggregates and structures information across entertainment, finance, and technology.

The DropThe platform maintains a knowledge graph of 1.83 million entities with 2.9 million typed relationships, covering movies, TV series, cryptocurrencies, public companies, and notable people. This Rust library provides full access to search, entity retrieval, and graph traversal.

# Installation

Rust 1.70+ (2021 edition) 
# Add to Cargo.toml
[dependencies]
dropthe = "0.1"
tokio = { version = "1", features = ["full"] }

# Or via cargo
cargo add dropthe

# Quick Start

use dropthe::{Client, EntityType};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Client::new("your-api-key");

    // Search the 1.83M entity database
    let results = client
        .search("Inception")
        .await?;

    for entity in &results {
        println!("{} ({})", entity.name, entity.entity_type);
    }

    // Filter by type with pagination
    let movies = client
        .list_by_type(EntityType::Movies)
        .limit(25)
        .offset(0)
        .send()
        .await?;

    println!("Found {} movies", movies.len());
    Ok(())
}

# API Reference

ASYNC client.search(query: &str) -> Result<Vec<Entity>>

Full-text search across all entity types in the DropThe knowledge graph.

ASYNC client.get_entity(slug: &str) -> Result<Entity>

Fetch a single entity with full metadata, data fields, and graph relationships.

ASYNC client.list_by_type(entity_type: EntityType) -> QueryBuilder

Build a filtered query for movies, series, people, companies, or cryptocurrencies.

ASYNC client.get_links(entity_id: &str) -> Result<Vec<EntityLink>>

Traverse the knowledge graph to find related entities via typed relationships.
